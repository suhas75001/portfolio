# Portfolio

A modern, responsive portfolio website built with vanilla web technologies.

## Tech Stack
- HTML5 for structure
- CSS3 for styling (including Flexbox/Grid for layout)
- Vanilla JavaScript for interactivity
- No external frameworks or dependencies needed

## Quick Start
1. Clone the repo:
   ```bash
   git clone https://github.com/suhas75001/portfolio.git
   cd portfolio
   ```

2. Start the project:
   - Option 1: Open `index.html` directly in your browser
   - Option 2: Use VS Code with Live Server extension
     1. Install "Live Server" extension in VS Code
     2. Right-click `index.html` → "Open with Live Server"
     3. Site opens automatically in your default browser

## Project Structure
```
portfolio/
├── index.html      # Main page with portfolio content
├── stayle.css     # Styling (includes responsive design)
├── main.js        # JavaScript for interactions
├── images/        # Image assets and portfolio pictures
└── pdf/           # Downloadable documents (resume, etc.)
```

## Features
- Responsive design that works on mobile, tablet, and desktop
- Clean, modern UI with smooth transitions
- Portfolio project showcase
- Contact form
- PDF downloads (resume/CV)

## Customization
1. Edit `index.html` to update content
2. Modify `stayle.css` for styling changes
3. Update `main.js` for new interactions
4. Replace files in `images/` and `pdf/` with your own
